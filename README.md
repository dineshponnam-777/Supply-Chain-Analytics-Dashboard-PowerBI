# 📊 Supply Chain Efficiency & Inventory Optimization Dashboard

## 🔍 Project Overview

This project focuses on analyzing supply chain performance and optimizing inventory management using Power BI. The dashboard provides insights into inventory levels, stockouts, supplier performance, and demand forecasting.

---

## 📌 Key Features

* KPI Cards: Total Inventory, Stockout Count, Forecast Accuracy, On-Time Delivery, Inventory Turnover
* Demand Forecast vs Actual Analysis
* Supplier Performance Comparison
* Inventory Distribution by Product Category
* Delivery Status Analysis
* Interactive Filters (Month, Region, Category)

---

## 📈 Key Insights

* Inventory levels are efficiently managed with minimal stockouts.
* Forecast accuracy is moderate, indicating scope for improvement.
* On-time delivery performance needs enhancement.
* Supplier performance varies, with some suppliers requiring attention.
* Inventory turnover indicates effective stock utilization.

---

## 🛠 Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Excel

---

## 📂 Dataset

The dataset includes:

* Product Details
* Inventory Levels
* Supplier Information
* Demand Forecast & Actual Data
* Delivery Status

---

## 🚀 Outcome

This dashboard helps in:

* Improving supply chain efficiency
* Reducing stockouts
* Enhancing supplier performance
* Supporting data-driven decision making
